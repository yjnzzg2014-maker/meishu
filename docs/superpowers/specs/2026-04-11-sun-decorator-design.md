# 「太阳装饰家」互动教学应用设计

## 一、项目概述

**项目名称**：「太阳装饰家」互动教学应用

**核心功能**：学生使用平板，在四分之一区域内拖拽几何图形和自然元素放置在太阳周围，学习对称概念。教师可通过控制端切换「轴对称模式」和「四分屏镜像模式」。

**目标学生**：小学一年级（6-7岁）

---

## 二、视觉与交互设计

### 整体风格
- **风格**：卡通童趣
- **色系**：暖阳马卡龙（粉、黄、橙、淡蓝为主）

### 背景设计
- 柔和渐变（淡粉→淡橙→淡黄），营造温暖日出氛围

### 太阳设计
- 固定居中于圆形画布中央
- 圆形金色填充
- 呼吸式发光动画（发光强度在 0.6-1.0 之间脉动）
- 8条可调节光芒（数量可配置）

### 图形元素
- **基础几何**：圆形、三角形、正方形、长方形
- **自然元素**：花朵、树叶、星星、云朵、小鸟
- 圆润边角，填充马卡龙色，带轻微阴影增加立体感
- 尺寸：80-100px，便于一年级学生操作

### 图形库布局
- 位于左侧固定区域
- 2列网格阵列展示
- 点击图形即可添加到画布中心附近

### 触摸交互
- 单指拖拽：移动图形位置
- 双指旋转：改变图形角度
- 拖出画布区域：自动删除图形

---

## 三、画布与镜像系统

### 画布设计
- 以太阳为中心的圆形区域
- 半径约为屏幕宽度的 40%

### 镜像模式

#### 轴对称模式（默认）
- 以太阳为原点，水平 + 垂直双轴镜像
- 学生仅操作左下角四分之一区域
- 所有操作实时镜像到其他三个象限

#### 四分屏镜像模式
- 屏幕均分四等份（2x2 网格）
- 所有分区显示相同内容
- 学生可在任一区域操作

### 模式切换
- 由教师端控制，一键切换
- 学生端实时生效，无需刷新

## 四、系统架构

### 拓扑结构
```
┌─────────────────────────────────────────────────┐
│                  教师控制端                       │
│  ┌─────────┐  ┌─────────┐  ┌─────────────────┐  │
│  │ 全局开关 │  │ 镜像切换 │  │   实时预览     │  │
│  │ 开启/关闭 │  │对称/四分 │  │   学生画面     │  │
│  └─────────┘  └─────────┘  └─────────────────┘  │
└──────────────────┬──────────────────────────────┘
                   │ WebSocket (Socket.IO)
          ┌────────┴────────┐
          │   局域网服务器   │
          │   (Node.js)     │
          └────────┬────────┘
                   │ 广播/同步
    ┌──────────────┼──────────────┐
    ▼              ▼              ▼
┌───────┐    ┌───────┐    ┌───────┐
│ 学生1  │    │ 学生2  │    │ 学生3  │
│ 平板   │    │ 平板   │    │ 平板   │
└───────┘    └───────┘    └───────┘
```

### 部署方式
- 纯本地局域网运行
- 教师端和学生平板连接同一 WiFi 网络
- 教师端作为 WebSocket 服务器

## 五、技术实现

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 前端渲染 | HTML5 Canvas | 性能最优，适合平板触控 |
| 触摸交互 | Pointer Events API | 统一处理触摸和鼠标事件 |
| 实时同步 | Socket.IO | WebSocket 封装，支持房间管理 |
| 后端服务 | Node.js + Express + Socket.IO | 轻量级局域网部署 |
| 状态管理 | 内存存储 | 局域网场景无需持久化 |

### 端口分配
- 服务器默认端口：3000
- 教师端：http://教师IP:3000/teacher
- 学生端：http://学生IP:3000/student

## 六、组件清单

### 学生端组件
1. **图形库面板**（左侧固定区域）
   - 2列网格展示所有图形
   - 点击添加图形到画布

2. **圆形画布**（中央主区域）
   - 太阳居中显示
   - 学生操作的图形显示区域

3. **镜像覆盖层**（透明覆盖）
   - 轴对称模式下显示三个镜像区域
   - 四分屏模式下显示四宫格布局

4. **操作提示**（首次使用）
   - 简单的拖拽和旋转提示动画

### 教师端组件
1. **模式切换面板**
   - 轴对称模式按钮
   - 四分屏镜像模式按钮

2. **全局控制开关**
   - 启用/禁用学生端操作

3. **学生连接列表**
   - 显示已连接的平板数量
   - 踢出连接功能

4. **实时预览窗口**
   - 模拟学生端画面

## 七、事件同步协议

### 客户端 → 服务器
| 事件名 | 载荷 | 说明 |
|--------|------|------|
| `add_shape` | `{type, x, y, angle, color}` | 添加图形 |
| `move_shape` | `{id, x, y}` | 移动图形 |
| `rotate_shape` | `{id, angle}` | 旋转图形 |
| `delete_shape` | `{id}` | 删除图形 |
| `change_mode` | `{mode}` | 切换镜像模式 |

### 服务器 → 客户端
| 事件名 | 载荷 | 说明 |
|--------|------|------|
| `sync_state` | `{shapes[], mode}` | 同步完整状态 |
| `broadcast` | `{action, payload}` | 广播操作到所有学生 |

## 八、边界情况处理

1. **网络断开**：学生端显示重连提示，自动尝试重连
2. **图形重叠**：按添加顺序渲染，最新图形在上层
3. **画布边界**：图形拖出圆形边界时显示半透明提示，松开后自动删除
4. **多指操作**：仅响应单指拖拽和双指旋转，忽略多余手指

## 九、验收标准

1. ✅ 学生可在平板上流畅拖拽图形（60fps）
2. ✅ 轴对称模式下，图形自动镜像到四个象限
3. ✅ 四分屏模式下，四区域同步显示
4. ✅ 教师端可实时切换模式
5. ✅ 太阳有呼吸发光动画
6. ✅ 图形库包含至少 9 种不同图形
7. ✅ 一年级学生可独立操作
