# 小学美术项目式学习 - 天空单元

这是一个小学一年级美术项目式学习（PBL）教学资源，包含天空单元的完整PPT课件和AI生成插图。

## 项目结构

```
meishu/
├── slides/                 # 第一部分：天空单元概览
│   ├── imgs/             # AI生成的插图
│   ├── output/           # 生成的PPT文件
│   └── slide-*.js       # 各页幻灯片源码
│
├── slides2/              # 第二部分：课程结构详解
│   ├── imgs/            # AI生成的插图
│   ├── output/           # 生成的PPT文件
│   └── slide-*.js       # 各页幻灯片源码
│
└── public/              # Web项目文件（绘画工具）
```

## 设计规范

- **配色方案**: Vibrant Orange Mint（明亮活泼，适合小学教育）
- **主色调**:
  - 深蓝 `#023047` - 标题
  - 蓝色 `#219ebc` - 正文
  - 黄色 `#ffb703` - 强调
  - 浅蓝 `#8ecae6` - 卡片
  - 淡蓝 `#caf0f8` - 背景
- **风格**: Rounded & Spacious（圆角友好风格）
- **字体**: Microsoft YaHei (中文) + Arial (英文)

## 技术栈

- **PPT生成**: PptxGenJS
- **插图生成**: MiniMax Image-01 模型
- **版本控制**: Git + GitHub

## 使用说明

### 本地查看PPT

```bash
# 进入项目目录
cd meishu

# 查看第一部分PPT
open slides/output/presentation.pptx

# 查看第二部分PPT
open slides2/output/presentation.pptx
```

### 修改幻灯片

每页幻灯片为独立的JavaScript文件，使用PptxGenJS库生成：

```javascript
// 修改后重新编译
cd slides
node compile.js
```
